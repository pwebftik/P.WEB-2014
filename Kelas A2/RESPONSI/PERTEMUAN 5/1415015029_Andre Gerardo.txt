1. Dalam konteks website, HTML adalah data dari output PHP ketika selesai dijalankan. 
HTML mendefinisikan struktur dan, untuk beberapa kasus, juga untuk penampilan halaman web. 
Jika Anda belajar untuk membangun situs web, Anda perlu belajar HTML pertama.
Secara umum, setiap situs yang cukup besar untuk menerima lebih dari beberapa pengunjung per bulan tidak akan dibangun dengan hanya HTML. 
Anda mungkin menemukan situs HTML statis untuk bisnis lokal kecil, tapi situs yang menghasilkan pendapatan online tidak hanya menggunakan HTML.
Anda tidak akan menemukan situs manapun yang dibangun dengan hanya PHP, karena PHP dengan sendirinya tidak mampu rendering halaman web tanpa menggunakan HTML.
Anda akan menemukan layanan web yang menggunakan PHP + beberapa format output selain HTML, XML atau JSON seperti, tetapi mereka dirancang ketat untuk dibaca oleh program komputer lain, bukan oleh manusia.
Dalam kebanyakan kasus, Anda tidak bisa mengatakan dengan pasti apakah situs tertentu menggunakan PHP atau tidak. 
Bahkan jika URL berakhir di php, itu tidak selalu berarti situs tersebut menggunakan PHP (meskipun mungkin).

PHP akan berguna ketika akan bekerja dengan menggunakan database. 
Sebagai contoh, Anda dapat menyimpan semua account pengguna Anda dalam database dan ketika anggota memasuki username dan password yang benar PHP Program / script,
Anda akan mengambil rincian pengguna yang relevan dari database dan menampilkannya di browser pengguna.
Dengan html itu akan mungkin bagi setiap orang untuk melihat halaman yang sama, 
halaman tidak akan terlihat berbeda atau menampilkan informasi yang berbeda tergantung siapa yang melihat itu, semua orang cukup banyak mendapat sama.


1. PHP mirip dengan HTML. Karena PHP memiliki keduanya, HTML dan kode PHP.
2. HTML adalah bahasa Tag tapi PHP adalah bahasa scripting.
3. Output dari PHP adalah sebagian besar dalam kode HTML. Karena browser web hanya mengirim kode HTML dan kode PHP selalu di latar.
4. Sebenarnya halaman web membutuhkan dua angka, menambahkannya dan hadir untuk pengguna yang mudah dilakukan oleh PHP tapi yang tidak mungkin dalam HTML.

PHP adalah side-server.HTML merupakan Browser side-server.
HTML digunakan untuk membuat isi dasar halaman, Tapi php memungkinkan halaman untuk 'bereaksi' dengan kondisi, dan menyesuaikan HTML yang sesuai.

Kelebihan dari PHP dibandingkan bahasa pemrograman lain:
1.      Bisa membuat Web menjadi Dinamis.
2.      PHP bersifat Open Source yang berarti dapat digunakan oleh siapa saja secara gratis.
3.      Program yang dibuat dengan PHP bisa dijalankan oleh Semua Sistem Operasi karena PHP berjalan secara Web Base yag artinya semua Sistem Operasi bahkan HP yang mempunyai Web Browser dapat menggunakan program PHP.
4.      Aplikasi PHP lebih cepat dibandingkan dengan ASP maupun Java.
5.      Mendukung banyak paket Database seperti MySQL, Oracle, PostgrSQL, dan lain-lain.
6.      Bahasa pemrograman PHP tidak memerlukan Kompilasi / Compile dalam penggunaannya.
7.      Banyak Web Server yang mendukung PHP seperti Apache, Lighttpd, IIS dan lain-lain.
8.      Pengembangan Aplikasi PHP mudah karena banyak Dokumentasi, Refrensi & Developer yang membantu dalam pengembangannya.
9.      Banyak bertebaran Aplikasi & Program PHP yang Gratis & Siap pakai seperti WordPress, PrestaShop, dan lain-lain.
